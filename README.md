# BankingApp
Banking Application with Primary and Savings account.

# Technical Stack used to develop this app:
 - Spring-Boot
 - SpringWebMVC
 - Spring-data-jpa
 - Spring-security
 - Spring REST
 - Thymeleaf-template-engine
 - Bootstrap
 - MySQL
